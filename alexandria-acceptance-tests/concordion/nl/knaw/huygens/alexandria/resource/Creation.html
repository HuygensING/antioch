Creating external resources

Examples:

How can I create an external resource without specifying an ID?
When I /resources the following body: 
{
  "resource" : {
    "ref" : "http://www.example.com/titles/path/to/title/resource"
  }
}
then I should receive 201 Created and the response contains a Location header starting with https://{host}/resources/, followed by a well-formed UUID.

The response body is  .

How can I create an external resource with a specific UUID?
When I /resources/6e56c868-c0a9-11e4-a762-cb9a664afc01 the following body: 
{
  "resource" : {
    "id" : "6e56c868-c0a9-11e4-a762-cb9a664afc01",
    "ref" : "http://www.example.com/titles/path/to/title/resource",
    "provenance" : {
      "who" : "nederlab",
      "why" : "some reason",
      "when" : "2015-04-07T09:10:55.877Z"
    }
  }
}
then I should receive 201 Created and the response body is  .
Then /resources/6e56c868-c0a9-11e4-a762-cb9a664afc01 should return 200 OK with a response body containing 
{
  "resource" : {
    "id" : "6e56c868-c0a9-11e4-a762-cb9a664afc01",
    "ref" : "http://www.example.com/titles/path/to/title/resource",
    "^provenance" : "https://{host}/resources/6e56c868-c0a9-11e4-a762-cb9a664afc01/provenance"
  }
}
And /resources/6e56c868-c0a9-11e4-a762-cb9a664afc01/provenance should return 200 OK with a response body containing 
{
  "provenance" : {
    "who" : "nederlab",
    "why" : "some reason",
    "when" : "{date.beforeNow}",
    "^what" : "https://{host}/resources/6e56c868-c0a9-11e4-a762-cb9a664afc01"
  }
}
